Навчальний проект

1. Визначено структуру "Раціональне число", перевизначено для неї оператор виведення в потік, арифметичні оператори.
2. Екземпляри структур використано для арифметичних обчислень.
3. Результати виведено:
   - на консоль
   - до файла
   - в пам'ять (в рядок) - використано виведення до потоку, накладеного на рядок !!!
4. Прочитано об'єкт з рядка. Для цього використано потік, накладений на рядок
5. Випробувано "удосконалену" функцію введення послідовності чисел з потоку

6. Створено шаблон структури "Раціональне число", який використовує ієрархію характеристик, щоб запобігти підстановці нечислових типів.
